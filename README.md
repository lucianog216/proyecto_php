# 🐱‍👤 Pokédex PHP

Aplicación web desarrollada en PHP que consume la API pública https://pokeapi.co para mostrar información de un Pokémon.

## 🚀 Funcionalidad

- Búsqueda de Pokémon por nombre
- Muestra:
  - Nombre
  - Altura
  - Peso
  - Habilidades
  - Imagen

## 🛠 Tecnologías

- PHP
- HTML
- CSS
- API REST (PokeAPI)

## ▶️ Cómo ejecutar

1. Subir archivos a servidor (XAMPP, hosting, etc.)
2. Abrir en navegador:
   http://localhost/pokemon-php

## 🌱 Control de versiones

- Rama utilizada: feature/desarrollo
- Commits descriptivos

## 📌 Notas

- Se usa file_get_contents para consumir la API
- Manejo básico de errores implementado
